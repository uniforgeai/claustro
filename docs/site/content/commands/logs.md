# claustro logs

Display container logs.

## Usage

```bash
claustro logs [name] [flags]
```

## Flags

| Flag | Description | Default |
|------|-------------|---------|
| `--name` | Sandbox name | |
| `-f, --follow` | Follow log output | false |
| `--tail` | Number of lines to show | 100 |
