# claustro burn

Stop and remove a sandbox container. Preserves the image and cache volumes.

## Usage

```bash
claustro burn [name] [flags]
```

## Flags

| Flag | Description |
|------|-------------|
| `--name` | Sandbox name |
| `--all` | Burn all sandboxes for the current project |
