# claustro claude

Launch Claude Code inside a sandbox. Automatically starts a sandbox if none is running.

## Usage

```bash
claustro claude [name] [flags] [-- claude-args...]
```

## Flags

| Flag | Description |
|------|-------------|
| `--name` | Sandbox name |

## Examples

```bash
claustro claude
claustro claude api
claustro claude -- --model opus
```
